Toolchain to support compiling with msvc on windows.

Values Added to projectSettings:

This toolchain will add the following items to the csbuild.projectSettings.projectSettings class when it is the active toolchain:

  • msvc_version: int denoting the msvc version
  • debug_runtime: bool denoting whether to link against a debug version of the runtime
  • debug_runtime_set: bool denoting whether or not the debug runtime flag has been set
