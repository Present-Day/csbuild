chunked_build()

source code 

Prepares the files for a chunked build. This function steps through all of the sources that are on the slate for compilation and determines whether each needs to be compiled individually or as a chunk. If it is to be compiled as a chunk, this function also creates the chunk file to be compiled. It then returns an updated list of files - individual files, chunk files, or both - that are to be compiled.
